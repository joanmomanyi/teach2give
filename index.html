Question 1: FizzBuzz

Write a program that prints the numbers from 1 to 100.

  • For multiples of 3, print "Fizz";
  • For multiples of 5, print "Buzz";
  • And for numbers that are multiples of both 3 and 5, print "FizzBuzz".

Solution

//Loop through all numbers from 1 to 100 //Current number is assigned varible i for (let i = 1; i <= 100; i++) { //We use modulus division (remainder) //If the remainder === 0 means number //is divisible //If i is divisible by 3 & 5; print "FizzBuzz" if (i % 3 === 0 && i % 5 === 0) { console.log("FizzBuzz"); //Else if i is divisible by 3, print "Fizz" } else if (i % 3 === 0) { console.log("Fizz"); //Else if i is divisible by 5, print "Buzz" } else if (i % 5 === 0) { console.log("Buzz"); //Otherwise, print the number i } else { console.log(i); } }
