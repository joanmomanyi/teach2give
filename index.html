Question 1: FizzBuzz

Write a program that prints the numbers from 1 to 100.

  • For multiples of 3, print "Fizz";
  • For multiples of 5, print "Buzz";
  • And for numbers that are multiples of both 3 and 5, print "FizzBuzz".

Solution

//Loop through all numbers from 1 to 100 for (let i = 1; i <= 100; i++) { //Use modulus division (remainder) if (i % 3 === 0 && i % 5 === 0) { //If i is divisible by 3 & 5; print "FizzBuzz" console.log("FizzBuzz"); } else if (i % 3 === 0) { //If i is divisible by 3, print "Fizz" console.log("Fizz"); } else if (i % 5 === 0) { //If i is divisible by 5, print "Buzz" console.log("Buzz"); } else { // Otherwise, print the number i console.log(i); } }
